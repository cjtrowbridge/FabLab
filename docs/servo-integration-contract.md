# Servo Integration Contract

Servos are high-risk purchased components because they combine body geometry, shaft offset, horn sweep, center screw access, wire relief, torque limits, and service paths.

Agents must not:

- invent servo geometry
- center a shaft unless the profile says it is centered
- block horn sweep
- block center screw access
- trap wires
- remove service access

## Preferred Workflow

- Select or create a servo profile.
- Record source/provenance and measurement confidence.
- Model the body envelope, shaft offset, horn sweep, screw access, and wire relief.
- Generate a receiver test coupon when a receiver pocket is new or risky.
- Block fabrication-ready claims until high-risk dimensions are measured or physically tested.
